# 欢迎来到 XplorOne

欢迎来到 XplorOne 的公开 GitHub 产品入口。

这个仓库主要承接：

- 文档说明
- 版本说明
- 下载入口
- 路线更新
- 问题反馈
- 社区讨论

它不是完整开源代码仓库首页。

## XplorOne 是什么

XplorOne 是一款面向自由职业者、超级个体、顾问、独立开发者和小型工作室主理人的本地优先 AI 财务工作台。

它帮助用户理解收入、支出、现金流、预算、账户结构和财务变化，同时保持清晰的本地数据边界与写入边界。

## 如何提供有用反馈

一般问题、想法和开放式反馈请使用 Discussions。

可复现 bug 或安装问题请使用 Issues，并尽量提供：

- XplorOne 版本
- Windows 版本
- 问题发生位置
- 复现步骤
- 预期结果
- 实际结果
- 必要时提供脱敏截图或日志

公开发布前，请移除私人财务数据、API Key、本地 API Token、MCP Token、授权码和个人信息。
