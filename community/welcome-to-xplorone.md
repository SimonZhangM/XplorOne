# Welcome to XplorOne

Welcome to the public GitHub hub for XplorOne.

This repository is primarily a product hub for:

- documentation
- release notes
- downloads
- roadmap updates
- issue reports
- community feedback

It is not a full open-source code landing page.

## What XplorOne Is

XplorOne is a local-first AI finance workspace for freelancers, solopreneurs, consultants, indie makers, and small studio owners.

It helps users understand income, expenses, cash flow, budgets, accounts, and financial patterns while keeping local data and write boundaries clear.

## How to Give Useful Feedback

For questions, ideas, and open-ended feedback, use Discussions.

For reproducible bugs or installer problems, use Issues and include:

- XplorOne version
- Windows version
- where the issue happened
- steps to reproduce
- expected result
- actual result
- redacted screenshots or logs if helpful

Please remove private financial data, API keys, Local API tokens, MCP tokens, license codes, and personal information before posting.

## Contribution Boundary

This repository is proprietary and product-facing.

Pull requests are currently limited to documentation fixes, typo fixes, broken-link fixes, and public metadata improvements unless explicitly discussed first.
