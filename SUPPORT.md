# Support

This repository is the public product hub for XplorOne. Use the right channel so questions, bugs, and product feedback stay easy to follow.

## Questions and product feedback

Use GitHub Discussions for:

- general questions
- product feedback
- workflow suggestions
- roadmap discussion
- help choosing the right setup path

Discussions are the best place when the topic is exploratory or when you are not sure whether something is a bug.

## Bugs and actionable problems

Use GitHub Issues for:

- installer problems
- reproducible bugs
- broken documentation links
- clearly actionable product problems

When reporting an issue, include:

- XplorOne version
- Windows version
- what you expected to happen
- what actually happened
- steps to reproduce the problem
- screenshots or logs if they help explain the issue

## Security issues

Do not open a public issue for security-sensitive reports.

For security-related concerns, use the process described in `SECURITY.md` once it is available. Until then, contact:
