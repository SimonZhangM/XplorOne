# WAIC 项目计划书

这个目录用于放置 XplorOne 的 WAIC FutureTech / OPC 项目计划材料。

## 这是什么

PDF 是为 WAIC 相关评审和外部沟通准备的公开项目计划文档。

文件：

- [XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf](./XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf)

## 一句话定位

XplorOne 是一款面向自由职业者、超级个体、顾问、独立开发者和小型工作室主理人的本地优先 AI 财务工作台。

它重点帮助用户理解收入、支出、现金流、预算、账户结构和财务变化，而不是从云端优先工作流开始。

## PDF 包含什么

项目计划书可能包括：

- 产品定位
- 用户场景
- 本地优先财务工作流
- AI 助手边界
- 数据与隐私考虑
- 当前产品能力
- 未来方向和项目路线
