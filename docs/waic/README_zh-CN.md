# WAIC FutureTech OPC 材料

这个目录用于放置 XplorOne 的 WAIC FutureTech OPC 独立先锋挑战赛材料。

## 这是什么

PDF 是一份 9 页中文视觉路演材料，用于 WAIC FutureTech OPC 评审和外部沟通。

它更接近一份简洁的产品展示 / 参赛说明，而不是长篇技术白皮书。

文件：

- [XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf](./XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf)

## 核心表达

这份材料把 XplorOne 表达为：

> 面向一人公司、超级个体、自由职业者和独立工作者的本地优先 AI 财务工作台。

它的核心判断是：很多超级个体不是不会赚钱，而是看不清自己的财务状态：

- 收入来自多个平台
- 账户和资金分散
- 生活支出与经营支出混在一起
- 未来现金流风险难以及时判断

## PDF 主要讲什么

这份 PDF 重点呈现：

- 为什么 AI 时代的一个人也需要新的财务基础设施
- XplorOne 如何从普通记账走向经营判断
- 从记录、结构化、查询、分析、决策，再回到记账的财务闭环
- 当前桌面产品已经落地的状态，包括账本、账户、分类、流水、预算、图表、现金流、资产负债等
- 本地助手、AI 助手与外部 agent 接入之间的分工
- 本地优先的数据基础，包括本地账本存储和受控访问边界
- 面向 Codex、WorkBuddy 等外部 agent 的只读 MCP 接入
- XplorOne 已经从想法走到可运行桌面产品
- 面向需要财务清晰度和更强分析能力用户的产品与商业化逻辑
