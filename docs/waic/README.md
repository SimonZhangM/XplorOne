# WAIC Project Plan

This folder contains XplorOne's WAIC FutureTech / OPC project-plan material.

The PDF is a public project-plan document prepared for WAIC-related review and communication.

File:

- [XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf](./XplorOne-WAIC-FutureTech-OPC-Project-Plan.pdf)

## One-Line Positioning

XplorOne is a local-first AI finance workspace for freelancers, solopreneurs, consultants, indie makers, and small studio owners.

It focuses on helping users understand income, expenses, cash flow, budgets, accounts, and financial patterns without starting from a cloud-first workflow.

## What the PDF Covers

The project plan may include:

- product positioning
- user scenarios
- local-first finance workflow
- AI assistant boundaries
- data and privacy considerations
- current product capabilities
- future direction and project roadmap
